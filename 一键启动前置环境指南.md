# 一键启动前置环境指南

本文档详细说明使用 `start.bat` 一键启动 Flask 城市景点可视化系统所需的前置环境配置。

## 🔧 必需软件安装

### 1. Python 3.8+ 安装

#### 下载地址

- 官方下载: https://www.python.org/downloads/
- 推荐版本: Python 3.9 或 3.10

#### 安装步骤

1. 下载 Python 安装包
2. **重要**: 勾选 "Add Python to PATH" 选项
3. 选择 "Install for all users"
4. 完成安装

#### 验证安装

```cmd
python --version
pip --version
```

应显示类似输出：

```
Python 3.9.7
pip 21.2.3
```

### 2. MySQL 5.7+ 安装

#### 方案一: MySQL Community Server (推荐)

**下载地址**: https://dev.mysql.com/downloads/mysql/

**安装步骤**:

1. 下载 MySQL Community Server
2. 运行安装程序
3. 选择 "Server only" 或 "Developer Default"
4. 设置 root 用户密码为 `123456`
5. 启用 "Start the MySQL Server at System Startup"
6. 完成安装

**验证安装**:

```cmd
mysql --version
```

#### MySQL 服务验证

**检查服务状态**:

```cmd
net start mysql  # 启动MySQL服务
```

**测试连接**:

```cmd
mysql -u root -p123456 -e "SELECT VERSION();"
```

成功输出示例：

```
+-----------+
| VERSION() |
+-----------+
| 8.0.28    |
+-----------+
```

## 🚨 常见问题解决

### 问题 1: Python 未添加到 PATH

**症状**: `python --version` 显示 "不是内部或外部命令"

**解决方案**:

1. 重新安装 Python，确保勾选 "Add Python to PATH"
2. 或手动添加到系统环境变量：
   - Python 安装路径: `C:\Python39\`
   - Python Scripts 路径: `C:\Python39\Scripts\`

### 问题 2: MySQL 服务无法启动

**症状**: `net start mysql` 显示服务不存在或启动失败

**解决方案**:

1. 重新安装 MySQL
2. 检查端口 3306 是否被占用
3. 以管理员身份运行命令提示符

**注意**: 完成以上环境配置后，直接运行 `start.bat` 即可一键启动整个系统！
